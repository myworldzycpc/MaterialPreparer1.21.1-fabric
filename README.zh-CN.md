# Material Preparer

[![EN](https://img.shields.io/badge/English-blue.svg)](README.md)
[![中文](https://img.shields.io/badge/简体中文-blue.svg)](README.zh-CN.md)

一个 Minecraft Fabric 模组，帮助你自动收集和整理材料。

> [!WARNING]
> 
> 本模组是客户端侧模组，理论上无需服务器安装，但由于该模组部分功能较强的辅助性质，部分服务器可能会禁止使用。
>
> 使用前务必确保服务器允许使用此类功能，以免被封禁。
>
> 因使用本模组导致的一切封禁、损失或法律纠纷，**模组作者概不负责**。

## 功能介绍

### 📦 智能物品收集
- 配置需要的物品列表，自动从附近的箱子中收集
- 支持精确移动，只拿需要的数量
- 可以忽略背包中已有的物品，只收集缺少的部分

### 🗺️ 容器探索
- 自动探索附近的容器并缓存物品信息
- 支持黑名单标记，跳过不想打开的容器
- 支持输出容器标记，背包满了自动存进去

### 📋 多种导入方式
- 从 Litematica 原理图文件导入物品列表
- 游戏内文件选择界面，支持子文件夹浏览
- CSV 格式存储，方便手动编辑

### ⚡ 可配置的操作速度
- 可调节容器点击间隔
- 避免过快点击导致的问题

### 🎨 调试可视化
- 显示容器的调试边框
- 不同颜色区分不同类型的容器
- 方便查看哪些容器被探索过

## 安装方法

1. 安装 [Fabric Loader](https://fabricmc.net/) 和 [Fabric API](https://www.curseforge.com/minecraft/mc-mods/fabric-api)
2. 安装 [Mod Menu](https://www.curseforge.com/minecraft/mc-mods/modmenu) 和 [YACL](https://www.curseforge.com/minecraft/mc-mods/yacl)
3. 将本模组的 `.jar` 文件放入 `mods` 文件夹

## 使用说明

### 配置物品列表

1. 打开 Mod Menu
2. 找到 Material Preparer，点击「配置」
3. 在「物品列表」分类中添加需要的物品和数量
4. 点击保存

### 收集物品

1. 设置好物品列表
2. 确保附近有箱子，里面有你需要的物品
3. 按下「收集物品」
4. 模组会自动打开附近的箱子并收集物品

### 从 Litematica 导入

1. 将 `.litematic` 原理图文件放入 `schematics` 文件夹
2. 打开配置界面
3. 在「物品列表」分类中找到「从 Litematica 加载物品列表」
4. 在文件选择界面中选择要加载的原理图文件
5. 点击「加载」按钮

### 标记容器

**黑名单**：
1. 将准星对准要标记的箱子
2. 按下「切换黑名单」
3. 再次按下可以取消标记

**输出容器**：
1. 将准星对准要作为输出的箱子
2. 按下「切换输出容器」
3. 背包满了会自动存到输出容器中

## 配置说明

### 通用设置

| 选项 | 说明                        | 默认值 |
|------|---------------------------|-----|
| 最小点击间隔 | 容器点击之间的最小间隔（tick），0 = 不限制 | 0   |
| 始终快速移动 | 是否总是使用 Shift+点击快速移动       | 关闭  |
| 忽略已有物品 | 是否只收集背包中缺少的物品             | 关闭  |

### 容器管理

| 操作     | 说明                |
|--------|-------------------|
| 切换黑名单  | 将准星对准的方块添加到/移出黑名单 |
| 切换输出容器 | 将准星对准的方块设为/取消输出容器 |
| 清除所有标记 | 清除所有容器标记          |

### 调试

| 选项     | 说明          | 默认值 |
|--------|-------------|-----|
| 显示调试边框 | 是否显示容器的调试边框 | 关闭  |
